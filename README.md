## 📖 About the Project

ClimaQuest is more than a weather app — it's a comprehensive platform designed to answer the question, *"Will It Rain On My Parade?"* It leverages NASA's historical Earth observation data to help users plan outdoor activities, learn about weather science, teach kids about climate, and support professionals with location-based climate risk data.

### 🌍 The NASA Space Apps Challenge

Our challenge was to construct an app with a personalized interface that enables users to conduct a customized query to determine the likelihood of **"very hot," "very cold," "very windy," "very wet,"** or **"very uncomfortable"** conditions for any location and date, using Earth observation data.

> *"If you're planning an outdoor event—like a vacation, a hike on a trail, or fishing on a lake—it would be good to know the chances of adverse weather for the time and location you are considering."* — NASA Earth Science Division

ClimaQuest is our answer to this challenge, combining:
- ✅ **Trip planning** based on historical weather-probability data
- ✅ **Weather science & analytics** with real charts and prediction algorithms
- ✅ **An interactive learning path for kids** through story episodes and quizzes
- ✅ **A 10-day weather forecast** for the user's current location
- ✅ **An Engineering Hub** for climate-aware construction decisions
- ✅ **Dark/Light mode** across every page, saved in the browser

## 🌍 Our Vision, Mission & Goals

### 🎯 Our Vision

Our vision is to make climate and weather knowledge accessible to everyone, no matter their age or background. We aim to create a platform that not only provides accurate and reliable weather data but also helps users understand, learn, and plan their activities with confidence.

> *"We believe that understanding the weather is not only for scientists — it's for travelers, students, families, and professionals who rely on climate information in their daily lives."*

### 🚀 Our Mission

Our mission is to make climate awareness and planning an easy and engaging experience for everyone. We achieve this by combining scientific accuracy, modern design, and interactive learning tools within a single platform.

**We fulfill our mission by:**
- 🔬 Using trusted data sources, including NASA and verified weather APIs
- 🎨 Designing an intuitive and accessible user interface
- 📚 Providing educational materials for different age groups
- ✈️ Offering personalized travel insights
- 🌱 Encouraging environmental awareness

### 🎯 Our Goals

Our main goal is to create one unified platform where anyone can easily access everything they need to know about the climate and weather — all in one place.

**Through our platform, users can:**
1. 📅 View the weather forecast for their current location for up to 10 days
2. 🔬 Learn how experts predict the climate using scientific formulas and methods
3. 👨‍👩‍👧 Access educational content and safety tips for children
4. 🏗️ Understand how engineers choose the right materials based on local weather
5. ✈️ Select their destination and travel time to discover expected weather conditions

## 🚀 Getting Started

This is a static front-end project — no build step or package installation required.

### Quick Start

1. Download / unzip the project files
2. Open `index.html` directly in your browser

### Run with Local Server (Recommended)

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (live-server)
npx live-server
```

Then open:
```
http://localhost:8000
```

> ⚠️ **Note:** The project loads some libraries from CDNs (Chart.js, Leaflet, Font Awesome), so an internet connection is required while running it.
